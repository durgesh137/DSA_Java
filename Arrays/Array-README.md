# Arrays Module - README

## Overview
This module contains all Java array problems and example solutions. 
It is organized into different levels based on problem complexity.


## Problems covered so far
1. Find frequency of element in an array
2. Find the minimum maximum element in an array
3. Move all negatives to end of array
4. Find peak element within an array
5. Find subarray with given sum
6. Find the missing number in an array
7. Find the largest sum contiguous subarray
8. Rotate array by `k` times (currently only by 1)
